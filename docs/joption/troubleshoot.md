# 異常排除

## 程式無法連線到券商 API

1. 確認網路正常
2. 確認 API 金鑰沒有過期
3. 確認 IP 是否在券商白名單內
4. 查看 log 檔案的錯誤訊息

## 程式卡住不動

- 查看 CPU / 記憶體是否異常
- 重啟程式，部位不會受影響
- 若頻繁發生，考慮增加 `check_interval`

## 部位與預期不符

- 手動查詢交易所目前部位
- 對照 log 確認下單紀錄
- 如有異常，先手動平倉確保安全

## 錯誤代碼查詢

| 錯誤代碼 | 說明 | 解決方式 |
|----------|------|----------|
| E001 | API 連線失敗 | 檢查網路與 API 金鑰 |
| E002 | 保證金不足 | 補充保證金或縮小部位 |

> 遇到其他問題？請在 GitHub 開 Issue 描述症狀。
