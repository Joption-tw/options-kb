# 操作教學

## 環境需求

- Python 3.10 以上
- 券商 API 帳號與金鑰

## 安裝步驟

```bash
# 1. 下載程式
git clone https://github.com/YOUR_REPO/joption.git
cd joption

# 2. 安裝套件
pip install -r requirements.txt

# 3. 複製設定檔
cp config.example.yaml config.yaml

# 4. 編輯設定檔，填入 API 金鑰
# 5. 啟動程式
python main.py
```

## 第一次使用建議

1. 先用**模擬帳戶**測試
2. 部位從小口數開始
3. 觀察 log 確認程式正常運作

> 詳細步驟持續補充中。
