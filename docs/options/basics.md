# 基本概念

## 什麼是選擇權？

選擇權（Options）是一種衍生性金融商品，賦予持有人在特定日期或之前，以特定價格買入或賣出標的資產的**權利**（非義務）。

## 買權 vs 賣權

| | 買權（Call） | 賣權（Put） |
|--|--|--|
| 持有者有權 | 買入標的 | 賣出標的 |
| 獲利方向 | 標的上漲 | 標的下跌 |
| 最大損失 | 權利金 | 權利金 |

## 履約價（Strike Price）

選擇權合約中約定的買賣價格。履約價與市場價格的關係：

- **價內（ITM）**：對持有者有利的狀態
- **價平（ATM）**：履約價 ≈ 市場價格
- **價外（OTM）**：對持有者不利的狀態

## 到期日

台灣期交所選擇權（台指選擇權）每個月第三個星期三為結算日。
