# 希臘字母（Greeks）

## Delta（δ）

選擇權價格對標的價格變動的敏感度。

- 買權 Delta：0 ~ +1
- 賣權 Delta：-1 ~ 0
- Delta = 0.5 表示標的漲 1 點，選擇權漲約 0.5 點

## Gamma（γ）

Delta 對標的價格變動的變化速度。Gamma 越大，Delta 變化越快，風險越高。

- 到期日越近、越接近價平，Gamma 越大

## Theta（θ）

時間衰減值，每天損失的時間價值。

- 賣方收 Theta（每天賺時間價值）
- 買方付 Theta（每天損失時間價值）

## Vega（ν）

對隱含波動率變動的敏感度。

- Vega 為正 → IV 上升獲利（買方）
- Vega 為負 → IV 下降獲利（賣方）

## Rho（ρ）

對利率變動的敏感度，台指選擇權通常影響不大。
