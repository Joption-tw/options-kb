# 選擇權定價

## 影響選擇權價格的因素

1. **標的資產價格** — 與履約價的距離
2. **隱含波動率（IV）** — 市場預期的未來波動
3. **到期時間** — 剩餘天數越多，時間價值越高
4. **無風險利率** — 影響相對較小
5. **股息** — 股票選擇權需考慮

## 隱含波動率（IV）

隱含波動率是從市場選擇權價格反推出來的預期波動率，反映市場的恐慌程度。

- IV 高 → 權利金貴 → 賣方有利
- IV 低 → 權利金便宜 → 買方有利

## 時間價值衰減

選擇權越接近到期日，時間價值衰減越快，尤其最後一週衰減最劇烈。
