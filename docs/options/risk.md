# 風險管理

## 保證金

賣出選擇權需要繳交保證金，期交所會根據部位風險動態調整。

- 帳戶保證金不足時，券商可能強制平倉
- 建議保留至少 2 倍以上的備用保證金

## 停損原則

| 方法 | 說明 |
|------|------|
| 固定金額停損 | 虧損達到某金額即出場 |
| 固定倍數停損 | 虧損達到權利金收入的 N 倍即出場 |
| 技術面停損 | 標的突破關鍵價位即出場 |

## 黑天鵝事件

極端行情下，選擇權價格可能瞬間暴漲，雙賣部位面臨巨大虧損。應對方式：

1. 控制部位大小
2. 買入保護性選擇權（組成 Iron Condor）
3. 設定停損並嚴格執行
